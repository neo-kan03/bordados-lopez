📂 Prototipos de Interfaz (Tests de Index)

Este repositorio contiene diferentes versiones de archivos index que se han utilizado como pruebas para experimentar con distintos diseños y estructuras de interfaz.

El objetivo de este repositorio es comparar varias implementaciones y decidir cuál se adapta mejor al proyecto principal.

📌 Archivo principal

El archivo que se considera versión final o principal es:

👉 index.html

Este es el archivo que representa el diseño elegido y el que se utiliza como referencia para el desarrollo.

🧪 Archivos de prueba

Los demás archivos index presentes en el repositorio son versiones experimentales utilizadas para:

Probar diferentes estructuras de interfaz

Evaluar distintos estilos o layouts

Comparar alternativas antes de elegir la versión final

Estos archivos se mantienen en el repositorio como referencia del proceso de desarrollo.

🚧 Estado

Este repositorio forma parte de un proceso de experimentación y prototipado, por lo que algunos archivos pueden estar incompletos o no representar versiones finales.
